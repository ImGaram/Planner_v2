## 📄 소개
하루 일정을 작성하고 확인할 수 있는 서비스입니다.

다음과 같은 기능을 제공합니다.
- 오늘 일정을 작성하고 특정 날짜를 선택해 일정을 작성할 수 있습니다.
- 사용자가 다양한 카테고리를 만들어 생성한 일정에 적용해 다양한 용도(필터링)로 활용할 수 있습니다.
- 지금까지 사용자가 작성하고 완료해 온 일정들의 개수를 그래프로 확인할 수 있습니다.
- 사용자들의 취향과 야간 사용을 고려해 다크 테마를 지원합니다.
